File Handling in Python

Python provides built-in functions to handle files.

Opening a File

Use open() to open a file.

file = open('filename.txt', 'r')  # Open file in read mode

Reading a File

Use read(), readline(), or readlines() to read the contents.

content = file.read()  # Read entire file
line = file.readline()  # Read one line
lines = file.readlines()  # Read all lines into a list

Writing to a File

Use write() or writelines() to write to a file.

file = open('filename.txt', 'w')  # Open file in write mode
file.write('Hello, world!')

Closing a File

Use close() to close the file.

file.close()
