capitalize() Method

Basic Example

Code:

text = "hello world"\n capitalized_text = text.capitalize()\n print(capitalized_text)

Output:

Hello world

Intermediate Example

Code:

text = "python programming language"\n capitalized_text = text.capitalize()\n print("Capitalized:", capitalized_text)

Output:

Capitalized: Python programming language

Advanced Example

Code:

sentences = ["welcome to python", "let's learn", "capitalize method"]\n capitalized_sentences = [s.capitalize() for s in sentences]\n print(capitalized_sentences)

Output:

['Welcome to python', "Let's learn", 'Capitalize method']

Real-Time Scenario

Code:

# Automatically capitalize the first word in sentences of a document.\n document = "python is powerful. programming is fun. keep learning."\n sentences = document.split('. ')\n capitalized_document = '. '.join([s.capitalize() for s in sentences])\n print(capitalized_document)

Output:

Python is powerful. Programming is fun. Keep learning.

lower() Method

Basic Example

Code:

text = "HELLO WORLD"\n lower_text = text.lower()\n print(lower_text)

Output:

hello world

Intermediate Example

Code:

mixed_text = "PyThOn Is AwEsOmE!"\n lower_text = mixed_text.lower()\n print(lower_text)

Output:

python is awesome!

Advanced Example

Code:

texts = ["PYTHON", "ProGRamMING", "Is FUn"]\n lower_texts = [t.lower() for t in texts]\n print(lower_texts)

Output:

['python', 'programming', 'is fun']

Real-Time Scenario

Code:

# Convert user input to lowercase before storing in database to ensure case-insensitivity.\n user_input = "Python Enthusiast"\n lower_input = user_input.lower()\n print("Storing in DB:", lower_input)

Output:

Storing in DB: python enthusiast
