String Methods in Python

  • capitalize(): Converts the first character to uppercase
  • casefold(): Converts the string to lowercase
  • center(): Centers the string within a given width
  • count(): Returns the number of occurrences of a specified value
  • encode(): Encodes the string using a specified encoding
  • endswith(): Checks if the string ends with the specified value
  • expandtabs(): Sets the tab size of the string
  • find(): Finds the position of the first occurrence of a specified value
  • format(): Formats the string with specified values
  • index(): Finds the position of the first occurrence of a specified value
  • isalnum(): Checks if all characters are alphanumeric
  • isalpha(): Checks if all characters are alphabetic
  • isascii(): Checks if all characters are ASCII
  • isdecimal(): Checks if all characters are decimal
  • isdigit(): Checks if all characters are digits
  • isidentifier(): Checks if the string is a valid identifier
  • islower(): Checks if all characters are lowercase
  • isnumeric(): Checks if all characters are numeric
  • isprintable(): Checks if all characters are printable
  • isspace(): Checks if all characters are whitespace
  • istitle(): Checks if the string is a title
  • isupper(): Checks if all characters are uppercase
  • join(): Joins elements of a list into a string
  • ljust(): Left-aligns the string within a given width
  • lstrip(): Removes leading whitespace
  • partition(): Splits the string into a 3-tuple
  • replace(): Replaces a specified value with another value
  • rfind(): Finds the position of the last occurrence of a specified value
  • rindex(): Finds the position of the last occurrence of a specified value
  • rjust(): Right-aligns the string within a given width
  • rpartition(): Splits the string into a 3-tuple
  • rsplit(): Splits the string into a list
  • rstrip(): Removes trailing whitespace
  • split(): Splits the string into a list
  • splitlines(): Splits the string at line breaks
  • startswith(): Checks if the string starts with the specified value
  • strip(): Removes leading and trailing whitespace
  • swapcase(): Swaps case of all characters
  • title(): Converts the first character of each word to uppercase
  • upper(): Converts all characters to uppercase
  • zfill(): Pads the string with zeros
