String Methods in Python

List of String Methods

  • capitalize(): Converts the first character to uppercase
  • casefold(): Converts the string to lowercase
  • center(): Centers the string within a given width
  • count(): Returns the number of occurrences of a specified value
  • encode(): Encodes the string using a specified encoding
  • endswith(): Checks if the string ends with the specified value
  • expandtabs(): Sets the tab size of the string
  • find(): Finds the position of the first occurrence of a specified value
  • format(): Formats the string with specified values
  • index(): Finds the position of the first occurrence of a specified value
  • isalnum(): Checks if all characters are alphanumeric
  • isalpha(): Checks if all characters are alphabetic
  • isascii(): Checks if all characters are ASCII
  • isdecimal(): Checks if all characters are decimal
  • isdigit(): Checks if all characters are digits
  • isidentifier(): Checks if the string is a valid identifier
  • islower(): Checks if all characters are lowercase
  • isnumeric(): Checks if all characters are numeric
  • isprintable(): Checks if all characters are printable
  • isspace(): Checks if all characters are whitespace
  • istitle(): Checks if the string is in title case
  • join(): Joins elements of an iterable into a string
  • ljust(): Left-justifies the string within a given width
  • lstrip(): Removes leading whitespace
  • partition(): Splits the string into a tuple of three parts
  • replace(): Replaces a specified value with another value
  • rfind(): Finds the position of the last occurrence of a specified value
  • rindex(): Finds the position of the last occurrence of a specified value
  • rjust(): Right-justifies the string within a given width
  • rstrip(): Removes trailing whitespace
  • split(): Splits the string into a list
  • splitlines(): Splits the string at line breaks
  • strip(): Removes leading and trailing whitespace
  • swapcase(): Swaps case of all characters
  • title(): Converts the string to title case
  • zfill(): Pads the string with zeros on the left

1. capitalize()

Use Cases: Converts the first character of a string to uppercase.

Scenario: Used to format user input or titles where the first letter should be capitalized.

Examples:

Code Level Code Example Output
Basic 

text = 'hello'
result = text.capitalize()
print(result)
                        
Hello
Intermediate 

text = 'john doe'
result = text.capitalize()
print(result)
                        
John doe
Advanced 

text = 'the quick brown fox'
result = ' '.join(word.capitalize() for word in text.split())
print(result)
                        
The Quick Brown Fox
Real-time 

user_input = 'jane doe'
result = user_input.capitalize()
print(f"Hello, {result}!")
                        
Hello, Jane doe!

Code Level Code Example Output
Basic

7. expandtabs()

Use Cases: Sets the tab size of the string.

Scenario: Useful for formatting text that includes tab characters.

Examples:

Code Level Code Example Output
Basic

def format_table(data, tab_size=4):
    return '\n'.join(line.expandtabs(tab_size) for line in data)

print(result)

John    30     New York

8. find()

Use Cases: Searches for a specified value and returns the index of its first occurrence.

Scenario: Useful for locating the position of a substring within a string.

Examples:

Code Level Code Example Output
Basic 

text = 'hello world'

Intermediate

Advanced 

def find_substring(text, substring):
    index = text.find(substring)
    return index if index != -1 else "Not found"

result = find_substring('search for this', 'for')
print(result)
                        
10
Real-time 

log_entry = 'Error: Unable to connect to the server.'
