1. capitalize()

Use Cases: Converts the first character of a string to uppercase.

Scenario: Used to format user input or titles where the first letter should be capitalized.

Examples:

Code Level Code Example Output
Basic 

text = 'hello'
result = text.capitalize()
print(result)
                        
Hello
Intermediate 

text = 'john doe'
result = text.capitalize()
print(result)
                        
John doe
Advanced 

text = 'the quick brown fox'
result = ' '.join(word.capitalize() for word in text.split())
print(result)
                        
The Quick Brown Fox
Real-time 

user_input = 'jane doe'
result = user_input.capitalize()
print(f"Hello, {result}!")
                        
Hello, Jane doe!

2. casefold()

Use Cases: Converts the string to lowercase, more aggressively than lower().

Scenario: Useful for case-insensitive comparisons, especially with Unicode text.

Examples:

Code Level Code Example Output
Basic 

text = 'HELLO'
result = text.casefold()
print(result)
                        
hello
Intermediate 

text = 'Thérè'
result = text.casefold()
print(result)
                        
thère
Advanced 

text = 'Python programming'
search_term = 'PYTHON'
result = text.casefold().find(search_term.casefold())
print(result)
                        
0
Real-time 

email = 'USER@DOMAIN.COM'
normalized_email = email.casefold()
print(f"Your email is: {normalized_email}")
                        
Your email is: user@domain.com

3. center()

Use Cases: Centers the string within a given width.

Scenario: Useful for formatting text output in a console or report.

Examples:

Code Level Code Example Output
Basic 

text = 'hello'
result = text.center(10)
print(result)
                        
  hello
Intermediate 

text = 'Python'
result = text.center(20, '-')
print(result)
                        
------Python-------
Advanced 

def pretty_print(text, width):
    return '\n'.join(line.center(width) for line in text.splitlines())

text = 'Header\nSubheader\nContent'
result = pretty_print(text, 50)
print(result)
                        

                  Header                  
               Subheader               
                 Content
Real-time 

invoice_number = '123456'
formatted_invoice = invoice_number.center(20, '0')
print(f"Invoice Number: {formatted_invoice}")
                        
Invoice Number: 00000001234560
