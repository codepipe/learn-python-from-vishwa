Python Data Types

Python supports various data types. The primary data types include:

  • int: Integer values
  • float: Floating-point numbers
  • str: Strings
  • list: Ordered, mutable sequences
  • tuple: Ordered, immutable sequences
  • dict: Dictionaries (key-value pairs)
  • set: Unordered collections of unique items

Each data type has its own methods and operations associated with it.
