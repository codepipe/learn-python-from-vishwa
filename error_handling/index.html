Error Handling in Python

Error handling in Python is done using try, except, else, and finally blocks.

Try and Except

Use try to test a block of code, and except to handle exceptions.

try:
    # Code that may cause an exception
except SomeException:
    # Code that runs if an exception occurs

Else and Finally

else runs if no exception is raised, and finally runs regardless of whether an exception occurred or not.

try:
    # Code that may cause an exception
except SomeException:
    # Code that runs if an exception occurs
else:
    # Code that runs if no exception occurs
finally:
    # Code that always runs
