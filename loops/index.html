Loops in Python

Python supports different types of loops, which are used to execute a block of code multiple times.

For Loop

The for loop is used to iterate over a sequence (like a list, tuple, string, or range).

for item in sequence:
    # Execute code

While Loop

The while loop executes as long as the condition is true.

while condition:
    # Execute code
